### What We Build

**[Butler](https://github.com/butlerdotdev/butler)** is an open source platform for provisioning and managing Kubernetes clusters at scale. Hosted control planes via [Steward](https://github.com/butlerdotdev/steward), built-in IPAM, multi-tenancy with team isolation, and a full addon ecosystem. One CRD creates a complete cluster.
